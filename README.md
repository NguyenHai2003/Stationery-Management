📦 Stationery-Management
Hệ thống quản lý văn phòng phẩm bao gồm server Spring Boot và ứng dụng Android (Java), hỗ trợ đặt hàng, quản lý kho và thanh toán.

🚀 Tính năng chính
✅ Đặt hàng văn phòng phẩm qua ứng dụng Android

✅ Quản lý kho sản phẩm theo thời gian thực

✅ Tích hợp cổng thanh toán MoMo

✅ Giao tiếp giữa client và server thông qua RESTful API

✅ Lưu trữ dữ liệu bằng MySQL

🛠️ Công nghệ sử dụng
Java (Android)

Spring Boot (Backend)

MySQL (Database)

Retrofit2 (Gọi API từ Android)

MoMo Payment Gateway

📱 Mô tả hệ thống
Ứng dụng Android cho phép người dùng duyệt sản phẩm, thêm vào giỏ hàng, chọn địa chỉ giao hàng và thanh toán.

Spring Boot API xử lý nghiệp vụ, lưu trữ dữ liệu và kết nối với MoMo để xử lý thanh toán.

CSDL MySQL lưu trữ thông tin sản phẩm, đơn hàng, người dùng, và chi tiết giỏ hàng.
